# Investment Journal (Decision Logs)

This directory contains records of key investment decisions made by the AI Fund Manager (CIO).

## Log Format
Each log entry should be named `YYYY-MM-DD-ticker-action.md` and include:
- **Action:** Buy / Sell / Hold / Rebalance
- **Ticker:** Stock symbol
- **Thesis:** Why this action is being taken (Fundamental/Technical reasons)
- **Risk Assessment:** Potential downsides or kill conditions
- **Execution Details:** Price, Size, etc.

## Purpose
To maintain transparency and allow for post-investment analysis (Post-mortem) to improve the AI's strategy over time.
